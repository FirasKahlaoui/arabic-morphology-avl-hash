# Arabic Morphology System

A C++ application using AVL trees and hash tables to model Arabic morphology (roots, schemes, and derivations).

## Project Structure

- `src/`: Source code (.cpp)
- `include/`: Header files (.h)
- `tests/`: Unit tests
- `data/`: Input data files
- `docs/`: Documentation
- `build/`: Build artifacts

## Build Instructions

```bash
make
./build/ArabicMorphology
```
